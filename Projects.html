Programming Languages Utilized: C#, Python, Unity, SQL

Description: This project is my senior design project for school. It is a year long project to demonstrate the mastery of computer science and be able to graduate. The application itself is a language-learning game built in unity. It is a part of an exisiting backend written in Python utilizing the Flask library. The database it utilizes is a mySQL database.

My Responsibilities:

  • Learn the pre-exisiting business logic
  • Write the backend services we need
  • Write the database table(s) we need
  • Interaction with new and old backend APIs
  • Serializing the API responses into useable C# classes
  • Incorporating those C# classes in our Unity code to hold correct data and send correctly formatted data to the APIs
  • Support other aspects of the game that pertain to the business logic
  • Write static data that follows the business logic structure to yser in the tutorial function of our game

Challenges:

  • Working with different languages in the tech stack caused some of the JSON to be formatted differently, causing me to have to debug and have to manually change the strings to make them serialize correctly
  • Learning existing business logic purely from written documentation and no word of mouth made some important information about the system get overlooked
  • Working with WebGL in Unity means that certain libraries/namespaces that may be a better fit for the application are not usable as they may not be stable in WebGL (async functions)

Learned information:

  • Enhanced my pre-existing skills of working with different business logic
  • More python experience
  • Learned more about dev-ops and how to host using Digital Ocean
  • Learned that a SQL database can use JSON data objects to incorporate a noSQL, document based structure to it as well
